source

pub fn parse(&mut self) -> Result<(), WsdlError>

Parses the WSDL document the structure was created with and fills out the structure’s fields to the best of its ability. This is a necessary step in generating the description of a SOAP API and should be called almost always after creating a new structure.

Some special behavior should be noted:

  • When determining the WSDL document’s version, if there is an error or an unsupported is found, the parser will fall back onto version (WsdlVersion::Wsdl11)[1.1]
Example
use std::path::Path;
use soap_rs::wsdl::WsdlDocument;

let mut api_description = WsdlDocument::new(Path::new("test-data/wsdl_sample_1.xml"))
    .expect("Failed to create document");
if let Err(e) = api_description.parse() {
    eprintln!("Failed to parse WSDL: {}", e);
}
